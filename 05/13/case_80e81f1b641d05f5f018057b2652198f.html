臺灣臺北地方法院刑事判決　　　　　　　　　　　　八十八年度訴字第一七九三號
　　公　訴　人　臺灣臺北地方法院檢察署檢察官
　　被　　　告　甲○○
　　選任辯護人　葉子瑋律師
右列被告因常業詐欺等案件，經檢察官提起公訴（八十八年度偵字第一四九一七號）
，及併案（九十年度偵字第九八號、第三三四號及台灣高雄地方法院檢察署八十九年
度偵字第二六三五八號），本院判決如左：
    主      文
甲○○共同意圖為自己不法之所有，以詐術使人將本人之物交付，以之為常業，處有
期徒刑壹年貳月。
    事      實
一、己○○（通緝中，到案後另結）與甲○○二人原係夫妻，共同基於意圖為自己不
    法之所有，並以此為常業之犯意聯絡，以己○○名義先後在台北市○○○路○段
    九十六號九樓開設榮邦國際顧問有限公司（簡稱榮邦公司），高雄市前鎮區○○
    ○路九一號二十樓之四設立元邦國際顧問股份有限公司（簡稱為元邦公司）己○
    ○、甲○○二人分別管理榮邦、元邦公司，其等對於加拿大移民法規熟稔，並對
    外宣稱該公司係與溫哥華著名小額專業律師JOHN T. MARTIN簽署獨家代理權，專
    門辦理加拿大移民業務等語，且以小額金錢即辦妥移民為鉺，誘使他人心動委託
    代辦移民等事務後，再以移民等事務需長期審核及等待，俟一定時日之後，再向
    申辦者佯稱取得初步審核資格，需商業投資加幣十萬元即商業移民，倘未移民成
    功於二年期滿退款，使申辦移民者誤以為自己再投資可取得移民資格，且未辦成
    期滿亦可退還投資款，因之陷於錯誤，乃依己○○、甲○○所建議為商業投資而
    支付加幣十萬元，己○○、甲○○見申請移民者上鉤後，隨即偽稱為充分保障投
    資人金額，真正投資金額為加幣十餘元，其餘為公司借款名目，可避免因公司經
    營成敗而負擔盈虧，確實保障移民者其權益，部分申辦者信以為真，乃簽署同意
    書，而對於部分不熟稔外文客戶，則以辦理需要而要求其等在外文書類簽署等手
    法，嗣二年期滿，該等客戶見未取得加拿大移民資格，要求解約還款時，己○○
    、甲○○即藉詞稱外國投資機構商務發生危機，致一時之間無法取回投資款，現
    在加拿大打官司中為由推拖搪塞，以之為詐財手法。自民國八十五年間起，己○
    ○、甲○○即以上揭手法，詐騙癸○○等人，其犯罪之詳確時地、被害人詐欺金
    額等均詳如附表所示。
二、案經被害人癸○○訴請台灣台北地方法院檢察署檢察官偵查起訴及併案偵辦。
    理      由
一、訊之被告雖坦承於高雄為元邦公司經理，經營代辦移民加拿大之業務等情，惟否
    認有何犯行，辯稱：己○○開設榮邦公司時，伊尚未認識己○○，不可能與己○
    ○共同開設榮邦公司，至八十九年七月十五日向己○○將被告開除，足見被告對
    公司經營無置喙餘地。關於移民業務之經營實況是榮邦公司刊登廣告後，如有客
    戶洽詢，即由己○○指定部分客戶由被告負責接洽，其餘所有移民事務，均由己
    ○○自行處理。元邦公司負責的是蒐集客戶文件，打包後寄至台北公司辦理認證
    ，元邦公司對於移民事務部分，無處理權責。被告與己○○之夫妻關係係於榮邦
    公司設立後才開始，於榮邦公司倒閉前即結束，婚姻關係中二人尚爭吵，被告南
    下負責榮邦公司業務後，更與己○○形同分居，二人離婚後，即無連絡，被告並
    已再婚，早已無夫妻情誼，絕非假離婚。被告對移民事務中是否有不法情事，因
    不涉此項業務，無從知悉。元邦公司於八十八年五月間政府規定移民業務之公司
    須加註登記並繳納保證金後，即依法申請，並於八十九年四月獲准註冊登記，足
    見元邦公司確係正常經營之公司，所以才具有資格申請並獲准登記註冊，因此被
    告相信元邦公司係合法經營之正常公司，被告為業務經理，依公司指示出外招攬
    、接洽客戶，為職務上之本分，何罪之有，至於己○○是否有何不法，被告全然
    不知云云？
二、經查證人即曾於八十七年底起在元邦公司任職秘書一年二月之卯○○到庭結證稱
    ：元邦公司在高雄辦理移民或留學之案子，負責蒐集客戶之文件寄至台北公司辦
    認證，費用由經理（甲○○）向客戶收取。客戶若有問題時，如由甲○○簽約就
    由甲○○處理，如己○○簽約之客戶就由己○○去處理，不知道費用怎麼算，只
    記得元邦公司曾款一、二次到總公司去，約一、二十萬元，感覺上像是金錢調
    度，元邦公司帳由甲○○自己管，己○○約二、三個月到高雄一次，當客戶需要
    模擬英文對話時，曹小姐才會來高雄，當天來回，如果曹小姐出面時，甲○○就
    不用出面。客戶催件時，甲○○先以電話中與客戶連絡，如需進一步處理，他才
    會向台北總公司連絡，他打電話時都會把門關起來，總公司都是直接與甲○○連
    絡，在我們員工認知裏因為他們二人是夫妻，所以甲○○應該也是有主導權，我
    常接到客戶抱怨的電話，都是案件處理進度的問題，有對父子辦理技術移民，我
    聽到他們說之前公司有承諾他們可以辦理移民，但後來都沒辦成，所以他們到公
    司來理論，這個案子是甲○○接到，到八十八年二月我離職時，這件事還沒解決
    等語。證人為被告之秘書，知悉元邦公司之營業狀況，所證為其親自聞見之事實
    ，且無何利害關係，其證言無誤認或誣攀之虞，其證言洵可採信。被告於高雄負
    責元邦公司之業務時，其親自向客戶招攬業務，收取各項費用，並且處理與客戶
    之糾紛等情，又為其所是認，雖被辯稱其所收費用均入台北總公司帳戶或交給台
    北總公司云云，惟未舉證以實其說，已難採信，況其所辯核與證人卯○○所證不
    符，所辯無非卸責之詞不足採信。被告對如附件之告訴人等人所委辦之移民業務
    ，收取高額之費用及所謂之投資款，歷時數年未辦成，拒不退費，對所謂之投資
    亦提不出相關之文件以證明確有投資款匯入所投資公司之帳戶內，雖己○○曾提
    出部分之匯款單以證明確有將客戶所收之投資款匯資到加拿大所欲投資之公司，
    並稱現正與所投資之各加拿大公司在加拿大訴訟中云云，惟查己○○所提出之匯
    款單，或可證明其有匯出部分款項至加拿大，惟不能證明各該匯款是為何人所匯
    ，且受款人又非原先所告知告訴人其等所投資之公司，且己○○所謂在加拿大訴
    訟，下次開庭會提出相關之訴訟文書為證，惟嗣逃逸無，現正為本院通緝中
    ，上開匯款單僅能為己○○有將部份款項匯至國外之事證，不足以為被告有利之
    證明。被告如不知己○○經營移民事務有關移民加拿大或留學加拿大之應俱備之
    條件及資格，其何能向客戶推銷接洽業務，客戶若非經由甲○○告知可以辦理移
    民或留學，客戶豈甘心交付鉅額之投資款及手續費，兼以被告甲○○先則為己○
    ○經營之榮邦公司之業務經理，為其重要事業夥伴，嗣進而結為夫妻，於公於私
    關係均甚為親密，其豈有不知己○○雖經營代客戶申請移民及留學加拿大事務之
    公司，對於不合乎條件之客戶，故意隱瞞該客戶不合於加拿大有關移民或留學之
    規定之實情，而詐取高額之投資款及手續費等情之理，況本件受害客戶數十人，
    其中本件經由被告甲○○經手之客戶亦達十餘人，甚且被告於八十八年八月五日
    於偵查中出庭應訊後，其猶與己○○繼續詐取乙○○、辛○○等人之投資款及費
    用，其顯與己○○有犯意之聯絡及行為之分擔甚明，被告甲○○辯均不足採，此
    外其犯行亦經如附表所示之被害人指訴不移，被告甲○○對各被害人所提出之相
    關文件（包括契約、收據等）均為是認，是本件被告甲○○之犯行明確，洵堪認
    定。
三、核被告甲○○所為，其與己○○於八十五年間開始共同詐欺癸○○時起，（嗣於
    同年七月十二日與癸○○簽約而於翌日詐得定金加拿大幣三萬元，八十七年三月
    十日又詐得加拿大幣十二萬元）與己○○有犯意之聯絡及行為分擔而共同以詐欺
    為常業，而犯刑法第三百四十條之罪，至其他由己○○單獨接洽經手之部分，因
    尚無證據證明被告甲○○與之共犯，應與甲○○無關，併此明。爰審酌被告犯
    罪之動機、目的、手段、品行、智識程度、所生危害及犯罪後之態度等一切情狀
    ，量處如主文所示之刑。
據上論斷，應依刑事訴訟法第二百九十九條第一項，刑法第二十八條、第三百四十條
，罰金罰鍰提高標準條例第一條前段判決如主文。
本案經檢察官侯少卿到庭執行職務。

│    │Market Printers Ltd.』，其投資額亦轉至後者，進而於八十八年五月四│
│    │日欲以合約附件將原契約條款中有關癸○○所投資對象部份為一變更，從│
│    │而，依原合約書第十一條小額投資方案之商務中心名稱應係『Asterra B-│
│    │usiness Center』，惟被告己○○歷次提出之憑據上名稱先是『Asterra │
│    │International Property & Captial Ltd. 』，再變為『Asterra Enter-│
│    │prise Centres Ltd.』，進而更變更為『Market Printers Ltd.』，而陳│
│    │進來之投資對象係一再變換主體，癸○○覺察不合常理，經多方查證得知│
│    │加國政府規定參與投資係於投資人『取得PR後』二年內投入資金即可，而│
│    │每半年需提出投資報告說明等情均與雙方簽訂契約書與被告等口頭承諾之│
│    │內容不符，再者，加國卑詩省並不接受房地產等行業為企業移民計劃，故│
│    │投資商務中心仍無法取得企業移民資格，至此癸○○始知被告等辦理企業│
│    │移民以取消條件小額投資方案為幌子，實得遂其詐取癸○○財產之目的，│
│    │故於八十八年五月十三日向被告等發函表示撤銷該委託契約，並應返還陳│
│    │進來已給付之加幣壹拾伍萬元，惟被告等迄今拒不返還。              │
├──┼────────────────────────────────┤
│ 2. │八十八年九月六日被告己○○至台中市長榮挂冠酒店與辛○○之妻黃錦玉│
│    │面洽，揚稱榮邦公司為合法公司，且保證所委辦之加拿大公民入籍申請案│
│    │一定善盡完成，且提出公司執照等證明使黃錦至及辛○○不知有詐，信以│
│    │為真，辛○○遂於八十八年九月六日與榮邦公司己○○簽立委託合約書，│
│    │並交付面額新台幣十四萬九千三百八十元之支票給己○○。依約榮邦公司│
│    │在十五日內未完成律師公證則不得兌現該支票。嗣己○○向辛○○誆稱業│
│    │於同年九月二十五日完成律師公證，而由甲○○於同年十月二日至台中縣│
│    │豐原市○○○路四四○巷六號辛○○家中退還上開支票，換取現金，陳人│
│    │忠不知有詐而如數交付，惟被告等詐欺得逞後，竟不履約，經辛○○向被│
│    │告等所謂辦理公證與榮邦公司同為乙方之馬田律師事務所查證，該事務所│
│    │以傳真聲明該事務所未參與，不負任何責任，且被告等均避不見面，陳人│
│    │忠始知受騙。                                                    │
├──┼────────────────────────────────┤
│ 3. │於八十八年三月間甲○○利用被害人子○○急於申請移民加拿大之際，在│
│    │榮邦公司向子○○詐稱榮邦公司係加拿大政府企業移民計畫「小額投資方│
│    │案Clobal Cogenix Idustrinl Corp. 加拿大B.C. 省水力發電工程案」之│
│    │法代理商，只要渠出資購買Global Cogenix Idustrial Corp.水力發電廠│
│    │之股票，該公司即可代為向加拿大政府申請投資移民，並保證一定獲得通│
│    │過，經雙方協議後簽訂企業移民委託契約乙紙，言明由子○○繳交十萬元│
│    │之手續費及佰零萬玖仟元之投資款給榮邦公司委託該公司代為投資「│
│    │小額投資方案Global Cogenix Idustrial Corp.水力發電廠股票及辦理企│
│    │業移民，榮邦公司應於半年內取得面談通知，一年內辦妥並取得加拿大永│
│    │久居民簽證。子○○不知有詐，如數交付，惟己○○等人事後並未履約，│
│    │未替子○○送件，經數度向己○○質問，及透過管道查證後，始知受騙，│
│    │要求還款，己○○等人避不見面。                                  │
├──┼────────────────────────────────┤
│ 4. │於八十九年三月間因乙○○欲移民加拿大，乃依報紙廣告與元邦公司連絡│
│    │，由甲○○基於前揭常詐欺之犯意，代表該公司與之洽談，甲○○向王澤│
│    │元詐稱要以海外投資移民之方式辦理，要乙○○先辦理加拿大國家銀行之│
│    │基金而於八十九年三月六日在台南縣六甲鄉龜港材龜子港四十二號乙○○│
│    │之住處內，由甲○○蓋用己○○及元邦公司之印章與乙○○簽訂投資移民│
│    │委託契約乙紙，言明由乙○○繳交手續費及壹佰肆拾捌萬肆仟元之購買基│
│    │金投資款給元邦公司，委託該公司代為投資符合加拿大政府投資移民計畫│
│    │規定之「加拿大國家銀行移民基金」及代辦投資移民，元邦公司應於半年│
│    │內取得面談通知，一年內辦妥並取得加拿大永久居民簽證。惟元邦公司於│
│    │收取前開款項後，並未替委託購買基金，亦未將投資款存入該銀行，被告│
│    │等並拒不還款，置之不理。                                        │
├──┼────────────────────────────────┤
│ 5. │八十八年七月初蘇峰被報紙上己○○刊登榮邦公司代辦移民廣告所欺，│
│    │而與己○○及甲○○洽談委託代辦移民加拿大之業務，己○○等人以前揭│
│    │詐欺方法使蘇峰誤信榮邦公司為正派有效率之公司，不疑有詐而於同年│
│    │月九日與己○○於台中市○○路○段二十二號五樓蘇峰所主持之東方牙│
│    │科診所簽訂委託代辦移民投資加拿大契約，己○○承諾於半年月內蘇峰│
│    │至加拿大面談，一年內取得加拿大居民簽證，蘇峰因之支付顧問費七萬│
│    │元，購買基金頭期款六十六萬元及規費十一萬元給榮邦公司，蘇峰並每│
│    │月向甲○○詢問面談之日期，甲○○均以下個月搪塞，致蘇峰無法如期│
│    │完成移民，喪失移民之用意及時機，嗣榮邦公司之業務改由元邦公司承受│
│    │，惟元邦公司乃未履約，被告己○○、甲○○均置之不理，且拒不退款，│
│    │蘇峰始知受騙。                                                │
├──┼────────────────────────────────┤
│ 6. │以前揭投資移民之廣告，騙使王琇麗於八十八年六月二十九日至王琇麗家│
│    │中（高雄市○○街二十一號十樓）與甲○○簽立投資移民加拿大之合約，│
│    │並於簽約時支付加拿大移民頭期款新台幣八萬元基金頭期款新台幣六十六│
│  │萬元，基金尾款新台幣一百三十六萬八千二百五十元嗣未履約，置之不理│
│    │，且避不見面，拒不退還相關費用。                                │
├──┼────────────────────────────────┤
│ 7. │八十八年二月七日以前揭詐欺方法騙使丙○○於高雄縣鳳山市○○路八十│
│    │四巷三十二號與甲○○簽立委託辦理加拿大移民入籍之事務，丙○○當場│
│    │支付新台幣十萬元之支票，嗣都以各種理由推託不辦，又不退款，被告曹│
│    │鳳姿及甲○○且避不見面，丙○○始知受騙。                        │
├──┼────────────────────────────────┤
│ 8. │以同右方法騙使午○○於八十八年十二月七日在高雄縣鳥松鄉松埔北巷二│
│    │之八十一號午○○家中與甲○○簽定委託元邦公司辦理入籍加拿大公民事│
│    │務，並當場支付新台幣十萬元，八十九年五月二十三日午○○支付二期款│
│    │新台幣五萬元及梲金新台幣二萬元，嗣即以各種理由推託，實則未履約，│
│    │亦拒不退款，午○○始知受騙。                                    │
├──┼────────────────────────────────┤
│ 9. │以同右方法騙使戊○○於八十七年六月三十日在台南縣永康市○○路七十│
│    │五巷三十二弄十二號戊○○家中與甲○○簽立委託元邦公司代辦加拿大移│
│    │民事務，戊○○當場支付現金新台幣十萬元，八十七年九月十五日支付規│
│    │費及報到費十一萬八千一百七十元，甲○○收款後，未履約為戊○○辦理│
│    │，戊○○向己○○、甲○○催詢均無結果，置之不理，戊○○始知受騙。│
├──┼────────────────────────────────┤
│ 10. │以同右方法騙使寅○○於八十七年十一月二十八日在高雄市○○區○○街│
│    │三十之三號四樓寅○○家中與甲○○簽訂委託元邦公司代辦以企業移民方│
│    │式移民加拿大之契約，寅○○當場支付新台幣九萬元，八十八年四月十五│
│    │日寅○○又支付規費新台幣八萬一千二百三十八元，嗣甲○○即以各種理│
│    │由推託，實未辦理，嗣元邦公司並以甲○○已離職為由，置之不理，劉秀│
│    │卿始知受騙。                                                    │
├──┼────────────────────────────────┤
│ 11. │以相同手法騙使丁○○於八十八年一月七日在高雄市苓雅區○○○街二十│
│    │六巷三十七號其家中與甲○○簽訂委託元邦公司代辦丁○○以企業移民之│
│    │方式移民加拿大之業務之契約，丁○○並當場支付新台幣九萬元由甲○○│
│    │收受，嗣即以各種理由推託不辦，亦不退款，置之不理，丁○○始知受騙│
│    │。                                                              │
├──┼────────────────────────────────┤
│ 12. │以前揭廣告騙使辰○○於八十八年十二月三十日在高雄市○○路六巷十四│
│    │號辰○○家中簽訂委託元邦公司辦理以投資移民方式移民加拿大之事務之│
│    │之契約，辰○○並當場支付新台幣十二萬元五千元手續費及基金頭款加拿│
│    │大幣一萬元由甲○○收受。嗣辰○○因故無法移民，乃經由己○○之同意│
│    │下解除契約，己○○同意退還加拿大幣一萬元（手續費不退），惟嗣避│
│    │不見面，經辰○○打聽結果，得知早已有多人受騙，始知被騙。        │
├──┼────────────────────────────────┤

│    │號與甲○○簽立委託元邦公司代辦以技術移民方式移民加拿大之事務之契│
│    │約，巳○○當場支付新台幣七萬元，由甲○○收受，八十八年八月四日簡│
│    │秀英再支付新台幣七萬零八十三元，惟嗣後即置之不理，巳○○始知受騙│
│    │。                                                              │

│ 14. │以同右方法騙使丑○○於八十七年三月十二日在高雄市○○路一○七巷一│
│    │號十六樓之二與甲○○簽訂委託元邦公司代辦以技移民方式移民加拿大之│
