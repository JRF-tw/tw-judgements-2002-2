【裁判字號】 90,訴,3794
【裁判日期】 910404
【裁判案由】 建築法
【裁判全文】   
九十一年三月十四日辯論終結
　　原　　　告　甲○○
　　被　　　告　台北市政府工務局
　　代　表　人　陳威仁（局長）
　　訴訟代理人　乙○○
　　　　　　　　丙○○
　　　　　　　　丁○○
右當事人間因建築法事件，原告不服台北市政府中華民國九十年三月七日府訴字第九
○○二三七○五○○號訴願決定，提起行政訴訟，本院判決如左：
　　主  文
原告之訴駁回。
訴訟費用由原告負擔。
　　事  實
一、事實概要：
    原告所有台北市大安區市○○道○段七十六號二樓建築物，位於第四種住宅區，
    領有被告核發之六一使字第一一二一號使用執照，核准用途為「集合住宅」，經
    台北市政府建設局於民國八十九年十月十八日派員前往稽查，發現前揭建築物供
    何錦健經營金飾加工業務，乃以八十九年十一月六日北市建一字第八九二六八八
    七八○○號函移被告所屬建築管理處及相關機關依權責辦理。嗣經被告審認系爭
    建築物之使用人何錦健，未經許可擅自違規使用為經營金飾加工業務，違反建築
    法第七十三條後段規定，乃依同法第九十條第一項規定，以八十九年十一月二十
    八日北市工建字第八九三五一九二一○○號書函，處以所有權人即原告新臺幣（
    下同）六萬元罰鍰，並勒令停止違規使用。原告不服，提起訴願，遭決定駁回，
    遂提起行政訴訟。
二、兩造聲明：
    (一)原告聲明：請求撤銷原處分及訴願決定。訴訟費用由被告負擔。
    (二)被告聲明：如主文所示。
三、兩造之爭點：
    (1)原告起訴有無逾期？
    (2)原告有無將其所有台北市市○○道○段七十六號二樓建物，供訴外人何錦健從
      事金飾加工業？
　　(一)原告主張：
      (1)台北市政府建設局查報員於八十九年十一月份至原告住宅處勘查，當時查報
        人員於現場拍照存證，並未說明原由而導致認定上之差別，在此原告再次向
        貴院說明，原告與家人一直居住在台北市市○○道○段七十六號二樓，未曾
        搬離，並未違反建築法第七十三條之Ｈ分類（供特定人住宿之場所）。
      (2)原告配偶何錦健從事銀樓雇員，因八十九年初銀樓更換工作之設備一批，將
        設備一批全數暫寄放於原告住宅處。實際上並未在本住宅從事任何金飾加工
        之工作，至於加工之工具因屬私人之配備，不便放至銀樓，拿回住家，才導
        致查報人員誤解。
      (3)原告一直居住在此，在十一月查報人員勘查時仍以此地為住處並有傢俱（沙
        發、冰箱、衣櫃、廚房廚具、電視、臥室臥具等家用品）及小孩遊戲房皆可
        證明，如因查報上認定之誤解偏差而需原告繳納鉅額之罰款，實在無法讓原
        告信服。
    (二)被告主張：
      (1)按建築法第七十三條規定：「建築物非經領得使用執照，不准接水、接電或
        申請營業登記及使用；非經領得變更使用執照，不得變更其使用。」、第九
        十條規定：「違反第七十三條後段規定擅自變更使用者，處建築物所有權人
        或使用人新臺幣六萬元以上三十萬元以下罰鍰，並勒令停止使用。得以補辦
        手續者，令其限期補辦手續，不停止使用或逾期不補辦者得連續處罰。」。
        行政訴訟法第一百零六條第一項規定：「撤銷訴訟之提起，應於訴願決定書
        送達後二個月之不變期間內為之。‧‧‧」、第一百零七條第一項第六款規
        定：「原告之訴，有左列各款情形之一者，行政法院應以裁定駁回之。‧‧
        ‧六起訴逾越法定期限者。」。
      (2)查系爭建築物領有被告核發六一使字第一一二一號使用執照，核准用途為集
        合住宅，未經申請變更使用執照，即擅自違規使用為工廠，從事金飾加工業
        ，案經目的事業主管機關即台北市政府建設局勘查認定有違規使用作為工廠
        具體情事，並於八十九年十一月六日北市建一字第八九二六八八七八○○號
        函向被告查告在案。被告審認原告所有系爭建築物供使用之情形，已違反建
        築法第七十三條後段規定，依同法第九十條第一項規定，於八十九年十一月
        二十八日北市工建字第八九三五一九二一○○號書函，處所有權人即原告六
        萬元罰鍰並勒令停止違規使用，並無違誤。
      (3)次查系爭建築物位於第四種住宅區，按原告實際上供使用之情形，及目的事
        業主管機關即台北市政府建設局認定為工廠，係屬臺北市土地使用分區管制
        規則第五條規定，核與原告使用執照所載建築物用途之集合住宅，分屬不同
        組別；又按「建築法第七十三條執行要點」規定，原告實際從事之工廠業務
        屬Ｃ類（工業類）第二組（Ｃ—二），為供儲存、包裝、製造一般物品之場
        所，而集合住宅則屬Ｈ類（住宿類）第二組（Ｈ—二），為供特定人士長期
        住宿之場所。另據內政部八十八年七月十六日臺八八內營字第八八七三八六
        九號函釋說明（一）決議：「建築法第七十三條執行要點係將建築物用途依
        使用強度及危險指標分為九類二十四組，是建築物用途如有擅自跨類組變更
        ，始違反建築法第七十三條後段規定，而依同法第九十條第一項規定處罰‧
        ‧‧」在案。揆諸首揭規定，被告依建築法第九十條第一項規定，於八十九
        年十一月二十八日以北市工建字第八九三五一九二一○○號書函處所有權人
        即原告罰鍰並勒令停止違規使用，並無違誤。原告訴訟理由所述並未從事金
        飾加工乙節，係由目的事業主管機關即台北市政府建設局主政認定之，非如
        原告主觀認知逕指誤解所致。故行政訴訟理由不足採據。
      (4)再查原告不服台北市政府九十年三月七日府訴字第九○○二三七○五○○號
        訴願決定，提起撤銷訴訟日期，係於九十年九月七日，而原告收受台北市政
        府上揭訴願決定書日期為九十年四月二十四日，原告於起訴狀中已陳明。故
        依首揭規定原告應於收受訴願決定書之次日起二個月內之法定不變期間內，
        即九十年六月二十四日前提起行政訴訟，原告提起訴訟已逾法定期限，自非
        法之所許。
      (5)末查行政機關對人民不履行行政法上義務時，迫其履行義務或督促使其與履
        行義務同一狀態，所採取之方式與程度，均應符合比例原則，於能實現義務
        之方法中選擇對義務人侵害最小之方法實施之，不得逾必要之程度，此為行
        政程序法第七條所明定，故於適用建築法第九十條規定予以處分時，自宜參
        酌。系爭建築物既違反建築法第七十三條後段規定情事，依法自應適用建築
        法第九十條規定予以處分，且為阻遏系爭建築物繼續供違規使用，暨貫徹前
        揭法律規定並保護公益及考量該建築物之違法使用狀態與嚴重程度，被告依
        建築法第九十條第一項規定，按其所營事業獲利程度及影響層面所為之罰鍰
        額度，核與比例原則、依法行政原則相符，且為最低限額，併予敘明。
　　理  由
甲、程序方面：
    按「撤銷訴訟之提起，應於訴願決定書送達後二個月之不變期間內為之。」行政
    訴訟法第一百零六條第一項定有明文。本件原告於九十年三月十五日收受訴願決
    定書，有郵務送達證書附訴願卷可稽，計其提起行政訴訟之期間，自九十年三月
    十六日起算，至九十年五月十五日屆滿，原告於九十年四月三十日提起本件行政
    訴訟，有其訴狀加蓋本院收文日期戳附卷可按，顯未逾法定二個月期間。被告謂
    原告係九十年九月七日起訴，已逾法定二個月期間，其起訴不合法云云，尚有誤
    會，合先說明。
乙、實體方面：
一、按建築法第七十三條規定：「建築物非經領得使用執照，不准接水、接電或申請
    營業登記及使用；非經領得變更使用執照，不得變更其使用。」、第九十條規定
    ：「違反第七十三條後段規定擅自變更使用者，處建築物所有權人或使用人新臺
    幣六萬元以上三十萬元以下罰鍰，並勒令停止使用。得以補辦手續者，令其限期
    補辦手續；不停止使用或逾期不補辦者得連續處罰。前項擅自變更使用之建築物
    ，有第五十八條所定各款情事之一者，停止供水、供電或封閉、強制拆除。」又
    建築法第七十三條執行要點第一點規定：「依本法條規定，建築物非經領得變更
    使用執照，不得變更其使用。建築物使用應按其使用強度及危險指標分類、分組
    ，如附表一。附表一未列舉之使用項目，由直轄市、縣（市）主管機關依使用類
    組定義認定之，並定期每季報請中央主管建築機關備查。」
    附表一
    建  築  物  使  用  分  類
   ┌────┬──────┬────┬──────┬────────┐
   │類    別│  類別定義  │組  別  │  組別定義  │  使用項目列舉  │
   ├─┬──┼──────┼────┼──────┼────────┤
   │Ｃ│ 工 │  供儲存、  │ Ｃ—２ │  供儲存、  │  一般工廠、工  │
   │  │ 業 │  包裝、製  │        │  包裝、製  │  作場、倉庫    │
   │  │ 倉 │  造、修理  │        │  造一般物  │                │
   │  │ 儲 │  物品之場  │        │  品之場所  │                │
   │類│ 類 │  所        │        │            │                │
   │  │    │            │        │            │                │
   ├─┼──┼──────┼────┼──────┼────────┤
   │Ｈ│ 住 │  供特定人  │ Ｈ—２ │  供特定人  │  住宅、集合住宅│
   │  │ 宿 │  住宿之場  │        │  長期住宿  │                │
   │類│ 類 │  所        │        │  之場所    │                │
   │  │    │            │        │            │                │
   └─┴──┴──────┴────┴──────┴────────┘
二、查原告所有之台北市大安區市○○道○段七十六號二樓建築物，位於第四種住宅
    區，，核准用途為「集合住宅」，此有被告核發之六一使字第一一二一號使用執
    照影本附原處分卷可稽。又前揭建築物供何錦健經營金飾加工業務，經台北市政
    府建設局於八十九年十月十八日派員前往稽查查獲，此亦有「加強取締違章工廠
    勘查紀錄表」影本附原處分卷可考。依上開紀錄表記載略以：「‧‧‧工廠地址
    ：台北市大安區市○○道○段七十六號二樓‧‧‧負責人：現場負責人何錦健。
    ‧‧‧主要產品：金飾加工。二、違反工廠設立登記規則，是。‧‧‧備註：鼓
    風機四台、熔爐一台、空壓機一台、展延機一台。經何錦健簽名。」顯見系爭建
    築物之使用人何錦健，未經許可擅自違規使用為經營金飾加工業務，已違反建築
    法第七十三條後段規定，被告依同法第九十條第一項規定，處以所有權人即原告
    六萬元罰鍰，並勒令停止違規使用，揆諸首揭規定，並無違誤，訴願決定予以維
    持，亦無不合。
三、原告起訴意旨略稱：因建設局查報員於八十九年十一月份至原告住宅處勘查，當
    時查報人員於現場拍照存證，並未說明原由而導致認定上之差別，原告與家人一
    直居住於系爭建築物內，未曾搬離，並未違反建築法第七十三條之Ｈ分類（供特
    定人住宿之場所）；又原告之配偶何錦健從事銀樓雇員，八十九年初銀樓更換工
    作之設備一批，全數寄放於系爭建築物內，實際上並未在系爭建築物內從事任何
    金飾加工之工作；至於加工之工具因屬私人之配備，不便放置銀樓，才導致查報
    人員誤解云云。惟查原告之夫何錦健確實有在系爭建築物內從事金屬加工業務，
    事證明確，已如前述，且原告於本院審理時，亦自認趕工時偶而在系爭建築內從
    事金屬加工無誤，是原告之起訴主張，顯非可採。復查系爭建物係位於第四種住
    宅區，核准用途為「集合住宅」，係屬建築法第七十三條執行要點規定之Ｈ類第
    二組供特定人長期住宿之場所，自不得違規挪作他用，縱使偶而、局部挪作他用
    ，亦非法之所許。原告於該址未經申請變更使用執照，即供何錦健違規使用為經
    營金飾加工業務之場所（係屬Ｃ類第二組供儲存、包裝、製造一般物品之場所）
    ，其已違反建築法第七十三條後段規定甚明。被告衡酌其違規作為經營金飾加工
    業務場所使用之情節，依同法第九十條第一項規定，處以建物所有權人即原告新
    臺幣六萬元罰鍰，並勒令系爭建築物停止違規使用之處分，實無不合。
四、綜上所述，本件原告之起訴主張，並非可採。原處分及訴願決定，俱無不合，原
    告訴請撤銷，核無理由，應予駁回。
丙、據上論結，本件原告之訴為無理由，依行政訴訟法第九十八條第三項前段，判決
    如主文。
